Seccion #2 (Variables)

Declaración de Variables

Codigo Ej.

              	var a=8;
	           b = 10;
	           let y = 5;
              	}
Resumen

Pudes declarar de 3 maneras

var a=8.Esta sintaxis puede ser usada para declarar tanto variables locales como globales.

b=10. Esto siempre declara una variable global y no puede ser cambiada a nivel local.

let y = 5. Esta variable puede ser usada para declarar una variable local en un bloque de ámbito.

Evaluacion de Variables

Codigo Ej.

              	var a;
                console.log('El valor de a es ' + a); // El valor de a es undefined

                console.log('El valor de b es ' + b); // El valor de b es undefined
                var b;
                console.log('El valor de c es ' + c); // Uncaught ReferenceError: c no está definida
								
                let x;
                console.log('El valor de x es ' + x); // El valor de x es undefined

                console.log('El valor de y es ' + y); // Uncaught ReferenceError: y no está definida
                let y;
              	}
Resumen

Consideraciones

Una variable declarada usando la sentencia var o let sin asignarle un valor inicial tiene el valor undefined.

Al intentar acceder a una variable no declarada dará como resultado la excepción ReferenceError.

Se puede usar undefined para determinar si una variable tiene un valor.

Ámbito de una Variable

Codigo Ej.

              	if (true) {
			  var x = 5;
			  }
			  console.log(x); // x vale 5
              	if (true) {
                          let y = 5;
                         }
                console.log(y);  // ReferenceError: y no está definida
Resumen

Consideraciones

Cuando declaras una variable fuera de una función, esta es llamada variable global, porque esta disponible para cualquier otro código en el documento actual. Cuando declaras una variable dentro de una función, esta es llamada variable local, porque está disponible solo dentro de esa función.

en elcódigo el primer log registrará 5, porque el ámbito de x es la función dentro del cual se declara x, no el bloque, que en este caso es la sentencia if.

el anterior comportamiento cambia, cuando usamos la declaración let

Operadores y Aritmetica

Codigo Ej.

              	var x = 5;         // signa el valor de 5 x
                var y = 2;         // signa el valor de 2 a y
                var z = x + y;     // signa el valor de 7 a z (x + y)
		var w = x * y      //signa el valor de 10 a w
Resumen

Operadores

Los operadores aritméticos se utilizan para realizar operaciones aritméticas con números
