1. Que le falta al siguiente codigo para que no de un error?
if (time < "10") {
 saludo = "buen dia";
els {
 saludo = "buena noche";
}
 a) Quitar las comillas al 10
 b) Cerrar las llaves del primer IF
 c) Els es incorrecto y se debe cambiar por else
 d) Todas las anteriores
2. Que le falta al siguiente codigo para que no de un error?
for (i < 5; i++) {
 text += "The number is " + i + ""
}
 a) "" Es una exprecion no permitida
 b) Falta el ;
 c) Falta inicializar la variable i
 d) Ninguna de las anteriores

3. Que le falta al siguiente codigo para que no de un error?
function miFuncion(a, b) { // a,b son parametros
  return a x b // retorna la multiplicacion
}
 a) El operador arimetico de "a x b" No es el correcto
 b) "//" No hacen parte de la sintaxis de JavaScript
 c) Una function no tiene esa estructura
 d) Falta el ;
